# 森活營家_露營入口網站

一個方便尋找與預訂營地的平台，另可以購買商品、集點、參加活動等等，同時也提供營地主自行上架營地。

個人主要負責部分：商品細節頁、商品購物車、結帳流程、還有視覺設計，前端部分使用React，後端則使用Node與資料庫做連結。

程式檔案主要位於 Camp-fe 和 Camp-be 這兩個資料夾中。
